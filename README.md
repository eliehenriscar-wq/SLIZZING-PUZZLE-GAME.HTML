# 🧩 Sliding Puzzle

Un jeu de puzzle classique 15-pièces fait en HTML, CSS et JavaScript pur — sans aucune bibliothèque externe.

![Sliding Puzzle](https://img.shields.io/badge/jeu-puzzle-brightgreen) ![HTML](https://img.shields.io/badge/HTML-5-orange) ![CSS](https://img.shields.io/badge/CSS-3-blue) ![JS](https://img.shields.io/badge/JavaScript-ES6-yellow)

---

## 🎮 Kijan pou jwe / Como jogar

- Klike sou yon pyes ki **bò kaz vid la** pou deplase li
- Mete pyes **1 jiska 15** nan lòd, soti agòch anlè rive adwat anba
- Kaz vid la dwe **ann dènye** (kwen anba adwat)
- Klike **"Bat kat"** pou yon nouvo jwèt

---

## ✅ Fonksyon / Funcionalidades

- ✔️ Griy 4×4 (15 pyes + 1 kaz vid)
- ✔️ Konte mouvman
- ✔️ Kont tan (minutè)
- ✔️ Deteksyon viktwa otomatik
- ✔️ Bouton "Bat kat" pou melanje
- ✔️ Verifye si jwèt la ka rezoud (algorithme solvabilité)
- ✔️ Responsive — fonksyone sou telefòn ak òdinatè
